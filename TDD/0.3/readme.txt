Project Milestone v 0.3

1) A class structure of dealers is implemented.  Dealer is an abstract base class
   with the member function Deal() being pure abstract
2) A class structure of players is implemented.  
3) An apporpriate table is created through an Object Factory
4) Each kind of table creates a dealer and a set of players appropriate
   To their game.
5) Players and Dealers are managed by the table. 
6) No other functionality is implemented.
